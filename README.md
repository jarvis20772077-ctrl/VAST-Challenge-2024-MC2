⚓ VAST 2024 MC2：“猎鹰”(Falcon) 全维度情报分析系统
1. 项目概述 (Project Overview)
“猎鹰”系统是一款专为打击全球非法捕捞（IUU）而设计的可视分析平台。针对 VAST Challenge 2024 MC2 提供的复杂异构知识图谱数据，系统通过整合时空轨迹、商业关系、资金流向及数字取证四个维度，从海量碎片信息中精准挖掘以 SSEC (SouthSeafood Express Corp) 为核心的非法捕捞网络。

2. 技术架构 (Technical Architecture)
系统采用模块化设计，确保了从原始数据到情报产出的高效转化：
前端框架：基于 Python 3.10 + Streamlit，实现响应式 Web 交互。
数据引擎：使用 Pandas 进行高性能异构数据处理与清洗。
可视化库：利用 Plotly 处理复杂的动态时序图、旭日图、桑基图及树状图。
部署环境：支持本地化一键部署，适配大规模 JSON 知识图谱解析。

3. 核心功能模块 (Core Modules)
A. 智能嫌疑评分引擎 (Suspicion Scoring)
系统不再依赖人工盲目搜索，而是通过算法自动筛选高危目标。
指标一：禁区驻留时长。统计船只在生态保护区（如 Oceanus 保护区）的累计停留小时数。
指标二：信号黑洞频次。识别 AIS 应答器关闭超过 4 小时的异常断联点。
产出：生成“嫌疑排行榜”，引导分析师优先审查得分最高的目标。

B. 语义化时空行为追踪 (Temporal Analysis)
将抽象的坐标转化为具象的行为逻辑。
可视化：语义化甘特图（Gantt Chart）。
逻辑：通过颜色编码（红色：禁区，蓝色：合法港口，灰色：断联）直观展示船只的作业轨迹，揭示其“何时隐身、何时入侵”。

C. 商业资产与资金穿透 (Business Forensics)
针对“幽灵船”刻意隐藏所有权的特征，系统执行反侦察分析。
可视化：多级树状图（Treemap）。
逻辑：打破名义所有权（Ownership）的伪装，通过商业交易（Transaction）的资金流向追溯实际控制人。若多艘幽灵船的渔获最终均流向 SSEC，即可判定其关联性。

D. 物流黑产链推演 (Logistical Trace)
重构从“捕捞”到“销赃”的完整路径。
可视化：桑基图（Sankey Diagram）。
逻辑：关联“禁区作业记录”与“港口卸货记录”，推演非法渔获是如何在合法港口被洗白并进入市场的。

4. 关键算法与逻辑说明 (Key Logic)
数据汉化与清洗：通过映射字典将复杂的实体标签（如 Entity.Vessel）转化为直观的中文，并利用 .str.strip() 解决原始数据中的空格脏数据问题。
时间窗口匹配：基于 dwell（停留时长）计算 end_time（离开时间），从而在时间轴上精确还原船只的作业区间。
取证审计 (Digital Forensics)：通过核对元数据 _last_edited_by 发现人为篡改痕迹，并检索 aphorism 等字段挖掘隐藏的接头暗号。

5. 项目亮点总结 (Highlights for PPT)
多维关联：将物理轨迹与商业金融数据深度融合。
逻辑闭环：实现了从“异常检测”到“关联发现”再到“证据固化”的分析全流程。
直观易用：全中文化界面，配合语义化颜色预警，极大降低了情报分析的门槛。
